# E-Stagio 📋

O E-Stágio é uma plataforma digital desenvolvida para simplificar o gerenciamento de estágios, oferecendo um meio prático e centralizado de registro e acompanhamento das atividades de estágio. O sistema é totalmente responsivo, adaptando-se tanto a telas de computadores quanto de dispositivos móveis para proporcionar uma experiência de uso consistente em qualquer dispositivo.

Disponível online <a href="https://estagios.fsw-ifc.brdrive.net" target="_blank">aqui</a>


## Objetivo 🎯

Plataforma web para gerir, otimizar e automatizar os processos de estágios supervisionados no IFC Videira.

## Público-alvo 🧑‍🧑‍🧒

- Alunos do Curto Técnico em Informática.
- Professores do IFC Videira, principalmente quando atuando como orientadores ou supervisores de estágio.
- Técnico-administrativos em Educação (TAEs) do IFC Videira, principalmente quando atuando como orientadores ou supervisores de estágio.
- Servidores do Setor de Estágio do IFC Videira.
- Supervisores de Estágio das Empresas Concedentes de Estágio.

## Processo de Desenvolvimento ⚙️

Ao longo de 2024, como Projeto de Extensão do IFC-Videira intitulado "Uma ferramenta computacional para gerenciar os estágios supervisionados do curso técnico em Informática do IFC Videira e otimizar o fluxo de trabalho dos servidores envolvidos." sob a coordenação do prof. Wanderson Rigo e tendo o empenhado aluno <a href="https://github.com/BrunoPergher" target="_blank">Bruno Vinícus Pergher</a>, do curso de Computação, como bolsista desenvolvendo o projeto no seu Trabalho de Conclusão de Curso (TCC).

Descrição Técnica:
- Desenvolvido em Python
- Banco de dados PostgreSQL
- Executado em contêiner Docker
- Hospedado em um servidor Linux (BRDrive)


## Benefícios 🌟

- Economizar tempo e papel;
- Simplificar as tarefas burocráticas;
- Promover uma comunicação eficiente entre estagiários, supervisores e orientadores;
- Oferecer um acompanhamento contínuo das atividades dos estagiários, melhorando assim a gestão e o controle das atividades;
- Aperfeiçoar a seleção de orientadores e a formação de bancas de avaliação.
